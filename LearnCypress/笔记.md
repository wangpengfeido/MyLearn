
## 使用

### 文件夹结构
```
cypress
├─integration    # 测试代码所在
│ 
```

### 测试策略
1. 数据构造。可以使用````cy.exec()````，````cy.task()````，````cy.request()````等命令请求服务器，让服务器生成合适的测试数据。
2. 服务存根。也可以绕过服务器，自己构造数据。服务存根可以与服务器数据构造结合，先使用正常服务器数据进行一次完全的测试，再使用服务存根测试边界情况。
3. 登录。全覆盖时可以使用UI来进行登录。但更好的是绕过UI，用请求来进行登录。
